- **Databases:** MySQL (approximately 250 MySQL 5.7 servers)

- **What they did:** Migrated from MySQL 5.7 to MySQL 8.0 (Percona Distribution + XtraBackup), adopted ExpertOps (formerly Managed Services), implemented PMM
- **Key outcomes:**

  - Stabilized customer-facing latency
  - Reduced downtime incidents by 40%
