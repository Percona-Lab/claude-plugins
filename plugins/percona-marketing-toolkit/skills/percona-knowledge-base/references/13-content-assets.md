# Percona: Content Asset Library for Outreach

This file maps real, published Percona content assets to outreach topics and funnel stages. Every content-led CTA in a Groove flow must reference an asset from this library. Do not invent fictional content.

Last updated: March 2026

---

## How to use this library

1. Every Day 21 "content-led" email must reference a real asset from this library.
2. Every "Happy to share how..." CTA should map to a specific asset below. If the BDR says "happy to share how other financial services firms handle X," the Referenced Asset field must name the specific case study, blog post, or eBook the BDR will actually send.
3. If no asset matches the topic, use a softer CTA: "I can put together a quick summary of how we approach this" (BDR-delivered, no asset needed).
4. The Referenced Asset field under each touchpoint must include the asset name and URL from this library.
5. Do not invent fictional guides, whitepapers, or blog posts that do not exist.
